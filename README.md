📊 Sona Store Data Analysis (Excel Project)
📌 Project Overview---
This project focuses on analyzing retail sales data of Sona Store to identify sales trends, customer behavior, regional performance, and profitability. The analysis is performed using Microsoft Excel with the help of pivot tables, charts, filters, and conditional formatting to generate actionable business insights.

🎯 Objectives
Analyze overall sales and profit performance
Identify top-performing products and regions
Understand customer demographics (gender, age group, segment)
Study monthly and seasonal sales trends
Provide data-driven recommendations for business improvement

📂 Dataset Description
The dataset contains transactional retail data with the following attributes:
Order Date & Ship Date
Ship Mode
Product Name, Category & Sub-Category
Sales, Profit, Quantity
Region & State
Customer Segment

🧹 Data Cleaning & Preparation
Converted date columns into proper date formats
Removed unnecessary fields (Row ID, Postal Code)
Handled missing values using placeholders
Ensured correct numeric formatting for Sales, Profit, Discount, and Shipping CostRemoved duplicate records
Standardized product and category names

🛠 Tools & Techniques Used
Microsoft Excel
Pivot Tables & Pivot Charts
Conditional Formatting
Filters & Sorting
Data Cleaning & Validation

📈 Key Analysis Performed
Orders vs Sales trend analysis
Gender-wise sales comparison (Men vs Women)
Top 5 states by sales performance
Order status analysis
Age-wise customer distribution
Sales channel performance
Monthly sales trend analysis

🔍 Key Insights
Top-Selling Product: Kutra generated the highest revenue
Highest Profit Product: Ethnic Dress with minimal discount
Top Performing Region: Maharashtra
Lowest Performing Region: Uttar Pradesh
Best Sales Month: April
Women customers contributed a higher share of total sales compared to men

✅ Business Recommendations
Improve marketing strategies in underperforming regions
Optimize shipping costs for high-delivery-charge products
Focus on high-profit, low-discount products
Use seasonal trends for inventory and promotion planning
Diversify product offerings to reduce dependency on limited categories
